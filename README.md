#  Task Tracker

A simple task management application built with Laravel as an educational project.

## About

This is a beginner-level project built to practice Laravel fundamentals including authentication, authorization, resource controllers, and database filtering.

## Tech Stack

- PHP 8.2
- Laravel 12
- SQLite
- Blade Templates

## Installation
1. composer install
2. npm install
3. cp .env.example .env
4. php artisan key:generate
5. php artisan migrate
6. npm run dev
7. php artisan serve
